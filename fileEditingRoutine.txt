Overwrite/replace a file (heredoc → sudo tee). Refresh sudo first so no prompt:
- echo changeme | sudo -S -v && cat <<'EOF' | sudo tee /etc/myapp/config.conf >/dev/null

Append to a file (heredoc → sudo tee -a):
- echo changeme | sudo -S -v && cat <<'EOF' | sudo tee -a /etc/myapp/config.conf >/dev/null
# Extra settings
option_c = enabled
EOF

In-place single-line edit (sed -i, portable regex):
- echo changeme | sudo -S sed -ri 's|^[[:space:]]*option_b[[:space:]]*=.*$|option_b = 99|' /etc/myapp/config.conf
